# Finance Dashboard UI

A polished, centered React dashboard for personal finance overview.

## Features

- Professional centered dashboard layout with a dark, glassmorphism-inspired panel.
- Summary cards for balance, income, and expenses with vibrant gradient styling.
- Animated entrance and subtle card motion for a smoother user experience.
- Line chart and pie chart visualizations using `recharts`.
- Transactions table with search filtering and role-based add button.
- Local storage persistence for transaction data.

## How to Run

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

Create a production build with:

```bash
npm run build
```

The optimized output will be generated in the `build` folder.

## Tech Stack

- React
- Recharts
- Tailwind CSS and custom CSS styling
- Local Storage for persistence

## Notes

This dashboard is designed as a visually polished UI demonstration rather than a full finance application backend. It focuses on layout, charts, styling, and user-friendly presentation.
